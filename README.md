# Smart Task Manager

A full-stack Task Management Application built using React.js, Spring Boot, JWT Authentication, and MySQL.

## Features

* User Registration
* User Login
* JWT Authentication
* Create Tasks
* Update Tasks
* Delete Tasks
* Search Tasks
* Task Status Management
* Secure APIs using JWT

## Tech Stack

### Frontend

* React.js
* Axios
* CSS

### Backend

* Spring Boot
* Spring Security
* JWT Authentication
* Spring Data JPA

### Database

* MySQL

## Project Structure

smart-task-manager

├── backend

└── frontend

## Run Project

### Backend

cd backend

mvn spring-boot:run

### Frontend

cd frontend

npm install

npm start
